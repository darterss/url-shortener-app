URL Shortener App
Приложение для сокращения ссылок с аналитикой.
Полный стек: NestJS (Backend) + React (Frontend) + PostgreSQL + Docker.

Запуск программы:
git clone https://github.com/darterss/url-shortener-app.git
cd url-shortener-app
docker-compose up --build

Запуск тестов:
cd backend
npm run test
